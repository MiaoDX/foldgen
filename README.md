# foldgen

AI origami design — give an agent a FOLD-based folding sim as a "self-verify in a physical domain" testbed, but with **human-teachable diagrams** as the output, not benchmark scores.

Second case study of the roboharness paradigm ("give the AI eyes to verify itself in a physical domain").
